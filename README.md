# Счётчик кликов по ссылкам
Это репозиторий для SMM-сотрудников ООО "Терминальная стадия". Содержит функции сокращения ссылок и отображения количества переходов по ним.

### Как установить
Установите зависимости из requirements.txt, в корневом каталоге создайте .env файл с вашим токеном с clc.li в формате "API_KEY=...".

### Для чего нужен
При запуске запрашивает ссылку. При вводе полной ссылки, сокращает ссылку через сервис clc.li и выводит ее в консоль; при вводе сокращенной ссылки, консоль выведет количество переходов по ней.

### Цель проекта
Код написан в образовательных целях на онлайн-курсе для веб-разработчиков dvmn.org.
